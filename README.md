# Ai-Health-Assistant-Healora
AI  Health Assistant is a web-based healthcare application that uses AI to provide health guidance, symptom insights, medicine reminders, and health record management. Built with HTML, CSS, JavaScript, a backend API, database integration, and deployed on AWS.
# 🩺 AI Health Assistant

AI Health Assistant is a full-stack web application that helps users manage their health using Artificial Intelligence.

## Features
- 🔐 User Authentication (Login & Signup)
- 🤖 AI Health Chat Assistant
- 💊 Medicine Reminder
- 📊 Health Dashboard
- 📄 Health Report Upload & Analysis
- 👤 User Profile Management
- ☁️ AWS Deployment

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js

### Database
- MySQL and mongoDB

### AI
- Large Language Model (LLM API)

### Cloud
- AWS EC2
- AWS S3
- AWS RDS
